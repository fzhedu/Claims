Detailed Description

LogicalOperator is the base class of all logical operator. GetPlanContext() aims at maintaining the property of the data after completing every operator of plan. the property contains the output attributes, partition key, cardinality and communication cost. GetPhysicalPlan() adds current physical operator into the whole plan, result into a plan tree.
